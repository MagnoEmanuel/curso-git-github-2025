git init . - iniciar repositório
git status - mostra o diretório
git add nomedoarquivo - sinaliza o que vai commit
git commmit -m "msg" - commit
git log - mostra histórico de commit
git diff - mostra o que foi modificado no arquivo
git reset nomedoarquivo - remove do stage antes do commit
git checkout -b nomedabranch - cria nova branch
git merge nomedabranch - mesclar as branchs selecionadas -
(serve para trazer as alterações de uma branch para outra)

1. criar diretório
2. iniciar com git init .
3. criar arquivos
